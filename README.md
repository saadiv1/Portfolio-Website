# Portfolio Website

A personal portfolio website built with HTML, CSS, and JavaScript to showcase projects, background information, and contact details in a clean single-page layout.

## Features

- Responsive single-page portfolio layout
- Hero, About, Portfolio, and Contact sections
- Tabbed About section for skills and education
- Mobile navigation menu
- Project cards with external live-demo links
- Contact form connected to Google Apps Script
- Downloadable resume button

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Font Awesome

## Project Structure

```text
Portfolio Website/
|-- index.html
|-- styles.css
|-- app.js
|-- images/
```

## Getting Started

1. Clone or download the project.
2. Open the project folder in VS Code or your preferred editor.
3. Open `index.html` in your browser.

If you want live preview inside VS Code, you can use the Live Server extension and launch the site from there.

## Customization

- Update text content in `index.html`
- Adjust colors, layout, and responsiveness in `styles.css`
- Edit menu behavior, tabs, and contact form logic in `app.js`
- Replace images and resume file inside the `images` folder

## Contact Form

The contact form submits data through a Google Apps Script endpoint defined in `app.js`.

If you want to use your own form backend:

1. Create a new Google Apps Script web app or another form endpoint.
2. Replace the `scriptURL` value in `app.js`.
3. Test the form submission in the browser.
